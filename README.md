# ChatVerse — Real-time Chat Application

Real-time chat application with rooms, video calls, and music streaming.

## Features
- 💬 Group Chat Rooms (create & join)
- 🔒 Private Messaging (1-on-1 DM)
- ✅ Read Receipts
- 📹 Video Call (WebRTC)
- 🎵 Music Room (shared playlist)
- 🕐 Timestamps on all messages
- 🔔 Join/Leave notifications

## Tech Stack
- **Backend**: Node.js + Express + Socket.IO
- **Frontend**: HTML + CSS + JavaScript
- **Video**: WebRTC
- **Music**: HTML5 Audio API

## Run Locally
```bash
npm install
node server.js
# Open http://localhost:3000
```

## Testing
1. Open 2 browser tabs at `http://localhost:3000`
2. Login with different usernames
3. Create rooms, chat, send DMs, video call, listen to music!
